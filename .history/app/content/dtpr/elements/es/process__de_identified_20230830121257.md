name: De-identified
id: process__de_identified
description: Datos que se procesan para eliminar los valores de identificación, generalmente para proteger la privacidad. La deidentificación puede ocurrir en cualquier momento en un proceso de recopilación de datos, por ejemplo antes de que se almacene en una base de datos, o antes de que se publique. AquÌ lo definimos para significar después de que los datos ya hayan dejado un dispositivo de hardware, o después de que se haya almacenado en una base de datos.Aprende más [aquí](https://en.wikipedia.org/wiki/De-identification)
