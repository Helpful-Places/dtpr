description: La firma de códigos es el proceso de firma digital de ejecutables y scripts para confirmar al autor del software y garantizar que el código no ha sido alterado o dañado desde su firma. El proceso emplea el uso de un hash criptogrófico para validar la autenticidad y la integridad.Aprende más [aquí](https://en.wikipedia.org/wiki/Code_signing)
