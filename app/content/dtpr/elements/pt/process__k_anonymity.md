name: K-Anonimato
id: k_anonymity
description: O anonimato K é utilizado para garantir o anonimato quando se divulgam dados que, de outra forma, seriam pessoalmente identificáveis. Os atributos dos dados são suprimidos ou generalizados até que cada linha seja idêntica a pelo menos k-1 outras linhas. Neste ponto, diz-se que a base de dados é k-anónima.
