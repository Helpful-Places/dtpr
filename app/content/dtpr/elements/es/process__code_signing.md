id: code_signing
description: La firma de código es el proceso de firmar digitalmente ejecutables y scripts para confirmar el autor del software y garantizar que el código no ha sido alterado o corrompido desde que se firmó. El proceso emplea el uso de un hash criptográfico para validar la autenticidad y la integridad. Obtenga más información [aquí] (https://en.wikipedia.org/wiki/code_signing)
